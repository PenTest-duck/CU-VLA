# CU-VLA

Computer Use (CU) with Vision Language Action (VLA) model.

Applying first-principles thinking to break down the problem of computer use into real-time robotics-style visuomotor problem. Running at a high vision and control frequency (30+ Hz; matching or exceeding human capabilities), we can break down using a computer into its fundamental primitives (e.g. key press, key release, mouse down, mouse move, scroll), with the screen display as input.

Down the line we may fuse ideas of world models.

This repo serves as an experimental research side project performed with consumer hardware (MacBook Pro M1 2020, 8GB RAM).
